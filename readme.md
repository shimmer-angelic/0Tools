这一个针对商务成本管理报表的编制工具。主要包括以下功能：
一、成本预算报表
1、根据模板新建报表
2、对成本项进行自动分解匹配
3、根据项目编码特征，对金额进行自动汇总计算
4、生成四类要素人、材、机、专的预算量汇总。同时可选择根据单位工程分别汇总，还是全部一起汇总。
二、成本月报
1、根据模板新建报表
2、选择基础预算，自动导更新当前月报
3、自动建立报表的公式逻辑
4、生成四类要素人、材、机、专的预算量汇总。同时可选择根据单位工程分别汇总，还是全部一起汇总。
三、工程量计算模板
1、输入计算包括说明，在结果列自动生成计算结果
2、选择不同沟体类型，自动添加相关计算要素到当前模板中
3、根据需求、可按工程部位或清单项进行汇总
